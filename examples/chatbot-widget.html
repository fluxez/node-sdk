Fluxez Chatbot Widget

This page demonstrates the Fluxez chatbot widget. The widget is fully customizable and can be integrated into any website with just a few lines of code.

Features:

  • Beautiful, responsive chat interface
  • Customizable colors and positioning
  • Real-time message streaming
  • Mobile-friendly design
  • Easy integration with service_ or anon_ API keys
  • Context-aware (supports organizationId, projectId, appId)

Usage:

// Import the browser module import { loadChatbot } from '@fluxez/node-sdk/browser'; // Load the chatbot with your API key const widget = loadChatbot('service_your_api_key', { position: 'bottom-right', theme: { primaryColor: '#0891b2', secondaryColor: '#06b6d4' }, startOpen: false, organizationId: 'your-org-id', projectId: 'your-project-id', appId: 'your-app-id' });
